## How to run

```bash
# Clone
$ git clone https://github.com/matheus-neves/bootcamp-gostack-desafio-01.git

# Go into the repository
$ cd bootcamp-gostack-desafio-01

# Install dependencies
$ yarn

# Init server
$ yarn dev
```
